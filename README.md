# Adult Income Dataset Analysis (Pandas)

This project performs exploratory data analysis on the Adult Income dataset using Python and Pandas.

## Objectives

- Analyze demographic features such as age, gender, and race
- Compare income groups (>50K vs ≤50K)
- Study working hours across countries
- Create age groups and analyze income distribution
- Filter occupation groups based on statistical conditions

## Key Insights

- The proportion of individuals earning >50K varies significantly across marital status and age groups.
- Married individuals tend to have a higher proportion of high earners.
- Most high-income earners fall within the **adult (35–70)** age group.
- Average working hours are higher among individuals earning more than 50K.

## Tools Used

- Python
- Pandas
- Jupyter Notebook

Adult Income dataset.
